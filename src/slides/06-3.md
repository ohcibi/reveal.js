## Example

* Beispielprogramm "Count Words"

```python
def count_words(text):
    ret = make_dict
    words = text split(" ")
    i = 0
    while i lt(words length):
        word = words get_element(i)
        count = 1
        if ret contains(word):
            count = ret get(word)
            count = count add(1)

        ret update(word, count)

        i = i add(1)
    ret
```
