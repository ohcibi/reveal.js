## Example

* Beispielprogramm "Bubblesort"

```python
def bubblesort(array):
    n = array length
    while 1 lt(n):
        i = 0
        while i lt(n sub(1)):
            el1 = array get_element(i)
            el2 = array get_element(i add(1))
            if el2 lt(el1):
                array set_element(i, el2)
                array set_element(i add(1), el1)
            i = i add(1)
        n = n sub(1)
```
